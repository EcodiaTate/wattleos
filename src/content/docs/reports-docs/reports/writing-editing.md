# Writing and Editing Reports

Once a report is generated, teachers use the Report Editor to write narratives, review auto-populated data, and mark sections as complete. The editor provides a focused writing environment with clear progress tracking.

## Accessing the Editor

From the **Reports** list page, click on any report to open it. The editor loads showing the student's name and photo, the report's current status, the term label, the author, and a progress indicator showing how many sections have been completed.

## Editor Layout

The editor displays all sections from the template in order. Each section shows:

- **Section title** — The heading from the template (e.g. "Teacher Comments," "Attendance Summary")
- **Section type indicator** — An icon or label showing whether this is auto-populated, semi-auto, or manual
- **Content area** — Either the auto-populated data display or a text editor for manual sections
- **Completion checkbox** — A toggle to mark the section as reviewed and complete

## Working with Auto-Populated Sections

Auto-populated sections (Student Information, Mastery Summary, Mastery Grid, Attendance Summary) display their data in a read-only format. The data was pulled from WattleOS at the time of generation and is stored as a snapshot in the report.

These sections are marked as complete automatically. You do not need to do anything with them unless you want to review the data for accuracy. If you notice an error in auto-populated data (e.g. wrong class listed), correct the underlying data in WattleOS and regenerate the report.

## Working with Observation Highlights

The Observation Highlights section shows the most recent published observations for the student. Each observation displays a content snippet, the author's name, linked curriculum outcomes, and media count.

This section is populated automatically but left as incomplete because you should review which observations are most representative. You cannot add or remove observations from this section in the editor — the selection was made during generation based on the template's maximum observation count.

After reviewing, mark the section as complete to indicate you are satisfied with the observation selection.

## Writing Manual Sections

Manual sections (Narrative, Custom Text, Goals) provide a rich text area where you write your commentary. The text area shows the placeholder hint from the template (if configured) when empty.

Write your content, then mark the section as complete. If the template specifies a suggested minimum word count, you will see a word counter below the text area. This is guidance, not a hard requirement — you can mark a section complete regardless of word count.

Tips for writing effective narratives:

- Reference specific observations and mastery progress that are shown in the auto-populated sections above
- Use the student's preferred name
- Balance strengths with areas for growth
- Include concrete examples of the student's work and choices
- Frame goals in positive, actionable terms

## Saving Changes

The editor saves your changes when you click the Save button. Changes to narratives and completion status are persisted to the database. You can leave the editor and return later — your work is preserved.

The editor tracks whether you have unsaved changes and displays a visual indicator when content has been modified since the last save.

## Completion Progress

The progress indicator at the top of the editor shows your completion status: how many editable sections (narrative, custom text, goals, observation highlights) have been marked complete out of the total editable sections. Auto-populated sections are not counted in this metric since they require no teacher input.

A report at 100% completion means all teacher-editable sections have been marked as reviewed and complete. This does not mean the report must be submitted — it is a visual aid to help you track your progress across multiple reports.

## Editing After Submission

If a report has been submitted for review and is sent back to draft status, all sections become editable again. You can revise your narratives, update completion markers, and resubmit. The content from your previous draft is preserved — nothing is lost when a report is sent back.

## Permissions

**Manage Reports** — Required to open the editor and write report content. In practice, the teacher who generated the report is typically the one who writes it, but any user with this permission can edit any draft or in-review report for their school.
