# Keyboard Shortcuts

WattleOS supports keyboard shortcuts to help you navigate quickly without reaching for the mouse. These work from any page in the application.

## Global Shortcuts

**Ctrl/Cmd + K** — Open Quick Search. This is the fastest way to find anything in WattleOS. Type a student name to jump to their profile, type a page name (like "attendance" or "mastery") to navigate there, or type an action (like "new observation") to start it. Quick Search works like a command palette — start typing and results appear instantly.

**Ctrl/Cmd + N** — New Observation. Opens the observation capture form directly, no matter which page you are on. This is designed for the guide workflow: see a learning moment, press the shortcut, capture it immediately.

**Ctrl/Cmd + /** — Open Ask Wattle. Launches the documentation assistant chatbot. Ask any question about how WattleOS works and get an answer based on the platform documentation.

**Escape** — Close. Dismisses any open modal, panel, overlay, or dropdown. If the sidebar is expanded on mobile, Escape closes it. If a lightbox is showing an observation photo, Escape closes the lightbox.

## Tips for iPad Users

On iPad with an external keyboard, use the **Cmd** key instead of **Ctrl** for all shortcuts. If you are using the on-screen keyboard without a physical keyboard, these shortcuts are not available — use the floating action button (the + button visible on most pages) to create observations, and use the sidebar for navigation.

## Quick Search Details

The Quick Search dialog (Ctrl/Cmd + K) searches across:

- **Students** — by first name, last name, or preferred name
- **Pages** — by section name (Dashboard, Observations, Curriculum, Attendance, etc.)
- **Actions** — common tasks like "new observation," "take roll," or "write report"

Results appear as you type with no need to press Enter. Click a result or use the arrow keys and Enter to navigate. The search is scoped to items you have permission to access — if you cannot see the Students section, student results will not appear in Quick Search.
